### 4.2 Bundle 2 Implementation

#### G1: Branch Badge

```python
# In header rendering
def render_header(self) -> Panel:
    branch = self._get_branch()
    content = Text()
    content.append("⚡ ", style="white")
    content.append(branch, style="cyan")
    content.append(f" │ Plan: {self.plan_name}")
    return Panel(content, title="Orchestrator")

def _get_branch(self) -> str:
    result = subprocess.run(
        ['git', 'branch', '--show-current'],
        capture_output=True, text=True
    )
    return result.stdout.strip() if result.returncode == 0 else "detached"
```

### 4.3 Bundle 3 Implementation

#### O1: Retry Badge

```python
# In task rendering
def render_task(task: Dict) -> Text:
    text = Text()
    # ... task ID and description ...

    retry_count = task.get('retryCount', 0)
    if retry_count > 0:
        max_retries = 3
        style = "yellow" if retry_count < max_retries else "red"
        text.append(f" [Retry {retry_count}/{max_retries}]", style=style)
    return text
```

#### O3: Orchestrator State

```python
def _get_orchestrator_state(self) -> str:
    """Detect orchestrator state from registry or process."""
    registry_file = '.claude/orchestrator-registry.json'
    if os.path.exists(registry_file):
        with open(registry_file) as f:
            registry = json.load(f)
        instances = registry.get('instances', [])
        if instances:
            return "● RUNNING"
    return "◯ IDLE"
```
