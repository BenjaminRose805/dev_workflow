## Implementation Details

### Quick Win 1: Enable fzf Shell Integration
**Time:** 2 minutes

```bash
# Add to ~/.bashrc
cat >> ~/.bashrc << 'EOF'

# fzf shell integration
source /usr/share/doc/fzf/examples/key-bindings.bash
source /usr/share/doc/fzf/examples/completion.bash
EOF

# Apply
source ~/.bashrc
```

**Verification:**
- Press Ctrl-R → Fuzzy history search appears
- Press Ctrl-T → Fuzzy file finder appears
- Press Alt-C → Fuzzy directory navigation appears

### Quick Win 4: Fix Yazi Git Fetcher
**Time:** 3 minutes

```bash
# Edit ~/.config/yazi/yazi.toml
# Add at the end:
cat >> ~/.config/yazi/yazi.toml << 'EOF'

[[plugin.prepend_fetchers]]
id   = "git"
name = "*"
run  = "git"

[[plugin.prepend_fetchers]]
id   = "git"
name = "*/"
run  = "git"
EOF
```

**Verification:**
- Open yazi in a git repository
- Modified files show 'M' indicator
- Untracked files show '?' indicator

### Quick Win 9: Install and Configure Delta
**Time:** 5 minutes

```bash
# Install
cargo install git-delta

# Configure git to use delta
git config --global core.pager delta
git config --global interactive.diffFilter "delta --color-only"
git config --global delta.navigate true
git config --global delta.side-by-side true
git config --global delta.syntax-theme "Monokai Extended"
```

**Verification:**
- `git diff` → shows syntax-highlighted side-by-side diff

### Quick Win 10: Create Yazi Keymap
**Time:** 5 minutes

```bash
cat > ~/.config/yazi/keymap.toml << 'EOF'
# Scroll preview
[[manager.prepend_keymap]]
on   = ["<C-e>"]
run  = "seek 5"
desc = "Scroll preview down"

[[manager.prepend_keymap]]
on   = ["<C-y>"]
run  = "seek -5"
desc = "Scroll preview up"

# Quick escape
[[manager.prepend_keymap]]
on   = ["<Esc>"]
run  = "escape --search --visual"
desc = "Cancel and clear"
EOF
```

**Verification:**
- Open yazi, preview a file
- Ctrl-E/Ctrl-Y scrolls preview

## One-Command Quick Setup Script

```bash
#!/bin/bash
# quick-wins.sh - Apply all Tier 1 quick wins

echo "=== Applying Quick Wins ==="

# 1. fzf shell integration
echo "1. Enabling fzf shell integration..."
if ! grep -q "key-bindings.bash" ~/.bashrc; then
    echo 'source /usr/share/doc/fzf/examples/key-bindings.bash' >> ~/.bashrc
    echo 'source /usr/share/doc/fzf/examples/completion.bash' >> ~/.bashrc
fi

# 2-3. Micro settings
echo "2-3. Updating micro settings..."
mkdir -p ~/.config/micro
if [ -f ~/.config/micro/settings.json ]; then
    # Add to existing settings
    jq '. + {"diffgutter": true, "hltrailingws": true}' ~/.config/micro/settings.json > /tmp/settings.json
    mv /tmp/settings.json ~/.config/micro/settings.json
else
    echo '{"diffgutter": true, "hltrailingws": true}' > ~/.config/micro/settings.json
fi

# 4. Yazi git fetcher
echo "4. Fixing yazi git fetcher..."
if ! grep -q 'prepend_fetchers' ~/.config/yazi/yazi.toml 2>/dev/null; then
    cat >> ~/.config/yazi/yazi.toml << 'EOF'

[[plugin.prepend_fetchers]]
id   = "git"
name = "*"
run  = "git"

[[plugin.prepend_fetchers]]
id   = "git"
name = "*/"
run  = "git"
EOF
fi

# 5. Git aliases
echo "5. Adding git aliases..."
git config --global alias.st status
git config --global alias.co checkout
git config --global alias.br branch
git config --global alias.ci commit
git config --global alias.df diff
git config --global alias.lg "log --oneline --graph --decorate -20"

echo "=== Quick Wins Applied ==="
echo "Run 'source ~/.bashrc' to activate fzf integration"
```
