# 1.4 Workflow Gaps and Pain Points

## Critical Gaps (Blocking or Major Friction)

### 1. No Git TUI Client
**Impact:** HIGH
**Current State:** Using raw `git` CLI commands
**Pain Points:**
- No visual staging/unstaging
- No interactive rebase UI
- No visual diff review
- Slow commit workflow
**Solution:** Install lazygit

### 2. No Syntax-Highlighted File Viewing
**Impact:** HIGH
**Current State:** Using `cat` for file viewing
**Pain Points:**
- No syntax highlighting in terminal
- Hard to read code outside editor
- Poor integration with other tools
**Solution:** Install bat

### 3. tmux Has No Configuration
**Impact:** HIGH
**Current State:** Default tmux with no customization
**Pain Points:**
- Default keybindings are awkward (Ctrl-b)
- No visual status bar
- No session persistence
- No integration with other tools
**Solution:** Configure tmux properly

### 4. fzf Not Integrated with Shell
**Impact:** MEDIUM-HIGH
**Current State:** fzf installed but no shell integration
**Pain Points:**
- No Ctrl-R history search
- No Ctrl-T file search
- No Alt-C directory jump
- Missing out on key fzf features
**Solution:** Enable fzf shell integration

---

## Significant Gaps (Workflow Inefficiencies)

### 5. No Smart Directory Navigation
**Impact:** MEDIUM
**Current State:** Manual cd commands
**Pain Points:**
- Can't jump to frequently-used dirs
- No history-based navigation
- Slow project switching
**Solution:** Install zoxide

### 6. No Git Diff Pager
**Impact:** MEDIUM
**Current State:** Default git diff output
**Pain Points:**
- Poor diff readability
- No syntax highlighting in diffs
- Hard to review changes
**Solution:** Install delta, configure as git pager

### 7. Missing JSON/YAML Processing
**Impact:** MEDIUM
**Current State:** No jq/yq installed
**Pain Points:**
- Can't easily parse config files
- No structured data manipulation
- Manual JSON editing error-prone
**Solution:** Install jq, yq

### 8. No Tree View
**Impact:** LOW-MEDIUM
**Current State:** No tree command
**Pain Points:**
- Can't visualize project structure
- Harder to orient in new codebases
**Solution:** Install tree or use eza --tree

---

## Configuration Gaps

### 9. No Git Aliases
**Impact:** MEDIUM
**Current State:** No aliases configured
**Pain Points:**
- Typing full git commands is slow
- No shortcuts for common workflows
**Solution:** Add common git aliases

### 10. No ripgrep Configuration
**Impact:** LOW
**Current State:** Using defaults
**Pain Points:**
- No smart case by default
- No custom type definitions
**Solution:** Create .ripgreprc

### 11. vim/nano Not Configured
**Impact:** LOW
**Current State:** Default configs
**Pain Points:**
- Occasional need to use vim (git commits, etc.)
- No line numbers or syntax highlighting
**Solution:** Basic .vimrc with essentials

---

## Integration Gaps

### 12. micro-yazi Bidirectional Missing
**Impact:** MEDIUM
**Current State:** Can open files from yazi→micro, but not navigate from micro→yazi
**Pain Points:**
- Can't easily browse from editor
- filemanager plugin exists but not used
**Solution:** Configure micro filemanager or add yazi opener

### 13. No fzf-git Integration
**Impact:** MEDIUM
**Current State:** fzf doesn't know about git
**Pain Points:**
- Can't fuzzy-search branches
- Can't fuzzy-search commits
- Can't fuzzy-search changed files
**Solution:** Add fzf-git shell functions

### 14. gh CLI Underutilized
**Impact:** LOW
**Current State:** Only one alias (co)
**Pain Points:**
- Not using gh's full potential
- Could have more workflow aliases
**Solution:** Add more gh aliases/extensions

|----------|-------|-----------|

## Quick Wins (Easy + High Value)
1. Enable fzf shell integration (just source files)
2. Add basic git aliases
3. Install bat (single package)
4. Configure tmux prefix key

## Bigger Efforts (High Value but More Work)
1. Install and configure lazygit
2. Install delta + configure git pager
3. Full tmux configuration
4. fzf-git integration functions
