# 3.1 High-Value Missing Features Per Tool

This document synthesizes findings from Phase 2 to identify the most impactful missing features for each TUI tool in the development workflow.

## Evaluation Criteria

Features are evaluated based on:
- **Impact**: How much will it improve daily workflow?
- **Friction Reduction**: Does it eliminate a common pain point?
- **Usage Frequency**: How often would this feature be used?

---

## Tool-by-Tool Missing Features

### 1. Yazi (File Manager)

| Priority | Missing Feature | Impact | Description |
|----------|----------------|--------|-------------|
| **HIGH** | Git fetcher config | Critical | Git plugin installed but not showing file status - missing fetcher configuration |
| **HIGH** | Custom keybindings | High | No keymap.toml - missing quick access to common operations |
| **HIGH** | Lazygit integration | High | No way to access git TUI from file manager |
| **MEDIUM** | Bookmark system | Medium | No quick navigation to frequent directories |
| **MEDIUM** | Archive handling | Medium | No ouch.yazi for compression/extraction operations |
| **LOW** | Custom theme | Low | Default appearance, no visual customization |

**Top 3 Missing Features:**
1. Fix git fetcher configuration (5 min fix, enables git status in file list)
2. Create keymap.toml with essential shortcuts (enables Ctrl-g for lazygit, scroll controls)
3. Install lazygit.yazi plugin (seamless git workflow)

### 2. Micro (Text Editor)

| Priority | Missing Feature | Impact | Description |
|----------|----------------|--------|-------------|
| **HIGH** | Git integration | Critical | diffgutter disabled, no visual git status in editor |
| **HIGH** | Multi-language LSP | High | Only Python configured; missing bash, JSON, markdown |
| **HIGH** | Trailing whitespace | Medium | hltrailingws disabled - can't catch whitespace issues |
| **MEDIUM** | Snippets plugin | Medium | No code snippets for common patterns |
| **MEDIUM** | Spell checking | Medium | No aspell/misspell for comments/docs |
| **MEDIUM** | Modern colorscheme | Low | Default theme, not optimal for long sessions |
| **LOW** | Search highlighting | Low | hlsearch disabled |

**Top 3 Missing Features:**
1. Enable diffgutter (true) - instant visual git feedback
2. Expand LSP config: add bash-language-server, deno lsp for JSON/markdown
3. Enable hltrailingws (true) - catch whitespace before commits

| **LOW** | ripgrep config | Low | No .ripgreprc for smart-case, custom types |

**Top 3 Missing Features:**
