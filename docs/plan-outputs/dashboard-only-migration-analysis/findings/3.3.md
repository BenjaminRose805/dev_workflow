## Critical Missing Features for Dashboard-Only Migration

Based on analysis, these features would be BLOCKERS for CLI/TUI removal:

1. **Plan Creation** - Users cannot create new plans
2. **Task Implementation** - Cannot execute tasks without orchestrator
3. **Git Integration** - No commit/branch management
4. **Worktree Management** - Cannot run parallel plans
5. **Plan Lifecycle** - Cannot archive, rollback, or cleanup
6. **Template System** - Cannot use plan templates

## Recommended Migration Approach

1. **Dashboard remains read/control interface** - View plans, start/stop/pause orchestrator
2. **CLI remains authoring interface** - Create, edit, archive plans
3. **Bridge critical gaps** - Add explain, verify buttons to dashboard
4. **Accept limitations** - Git/worktree features remain CLI-only
