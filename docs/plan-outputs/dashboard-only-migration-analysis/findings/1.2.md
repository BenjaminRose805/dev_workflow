## Missing/Limited Features

Based on analysis, these features are NOT present in the dashboard:
1. **Create/Edit Plans** - Dashboard is read-only for plan content
2. **Manual Task Execution** - No way to run individual tasks
3. **Plan File Editing** - No markdown editor
4. **Git Integration** - No branch switching, commits
5. **Worktree Management** - Not exposed in UI
6. **CLI Command Execution** - No terminal interface
7. **Log Viewing** - No raw log access
8. **Configuration Editing** - No git-workflow.json editing
9. **Template Management** - No template creation/listing
10. **Multi-Plan Orchestration** - No orchestrate multiple plans at once
