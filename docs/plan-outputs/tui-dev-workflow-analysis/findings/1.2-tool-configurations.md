# 1.2 Tool Configurations

## Yazi (File Manager)

**Config Location:** `~/.config/yazi/`

### yazi.toml
```toml
[plugin]
prepend_previewers = [
  { name = "*.md", run = "glow" },
]
```
- Markdown files previewed with glow plugin

### init.lua
```lua
require("git"):setup()
```
- Git status integration enabled

### Installed Plugins
| Plugin | Source | Purpose |
|--------|--------|---------|
| git.yazi | yazi-rs/plugins:git | Git status in file list |
| glow.yazi | Reledia/glow | Markdown preview with glow |

## Micro (Text Editor)

**Config Location:** `~/.config/micro/`

### settings.json
```json
{
    "lsp.server": "python=pylsp",
    "lsp.autocompleteDetails": true,
    "lsp.formatOnSave": true
}
```
- LSP enabled for Python (pylsp)
- Autocomplete with details
- Format on save enabled

### bindings.json
| Key | Action |
|-----|--------|
| Alt-/ | Comment toggle |
| Alt-d | Go to definition (LSP) |
| Alt-f | Format (LSP) |
| Alt-k | Hover info (LSP) |
| Alt-r | References (LSP) |
| Ctrl-Space | LSP completion |
| F2 | Next bookmark |
| Shift-F2 | Previous bookmark |
| Ctrl-F2 | Toggle bookmark |
| F4 | Jump to tag |

### Installed Plugins

|--------|---------|

| bookmark | Bookmark support |
| editorconfig | EditorConfig support |
| filemanager | File browser sidebar |
| fzf | Fuzzy file finder |
| jump | Jump navigation |
| lsp | Language Server Protocol |
| manipulator | Text manipulation |
