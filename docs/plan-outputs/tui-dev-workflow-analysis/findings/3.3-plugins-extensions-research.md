## Yazi Plugins

### Essential Plugins (HIGH Priority)

#### 1. lazygit.yazi
**Purpose:** Access lazygit TUI from within yazi
**Repository:** yazi-rs/plugins:lazygit
**Status:** NOT INSTALLED

**Installation:**
```bash
ya pkg add yazi-rs/plugins:lazygit
```

**Configuration (keymap.toml):**
```toml
[[manager.prepend_keymap]]
on   = ["<C-g>"]
run  = "plugin lazygit"
desc = "Open lazygit"
```

**Requirements:** lazygit must be installed
**Impact:** HIGH - Seamless git workflow from file manager

## tmux Plugins

### Essential Plugins (HIGH Priority)

#### 1. TPM (Tmux Plugin Manager)
**Purpose:** Manage tmux plugins
**Repository:** github.com/tmux-plugins/tpm
**Status:** NOT INSTALLED

**Installation:**
```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

**Configuration (~/.tmux.conf):**
```
set -g @plugin 'tmux-plugins/tpm'
run '~/.tmux/plugins/tpm/tpm'
```

---

#### 2. tmux-sensible
**Purpose:** Sensible default settings
**Repository:** tmux-plugins/tmux-sensible
**Status:** NOT INSTALLED

**Installation (after TPM):**
Add to .tmux.conf:
```
set -g @plugin 'tmux-plugins/tmux-sensible'
```

**Features:**
- UTF-8 support
- Better history
- Faster key repetition
- Improved defaults

---

#### 3. tmux-resurrect
**Purpose:** Save and restore sessions
**Repository:** tmux-plugins/tmux-resurrect
**Status:** NOT INSTALLED

**Installation:**
```
set -g @plugin 'tmux-plugins/tmux-resurrect'
```

**Usage:**
- `prefix + Ctrl-s`: Save session
- `prefix + Ctrl-r`: Restore session

**Persists:** Windows, panes, layouts, directory, programs

---

#### 4. tmux-continuum
**Purpose:** Automatic session saving
**Repository:** tmux-plugins/tmux-continuum
**Status:** NOT INSTALLED

**Installation:**
```
set -g @plugin 'tmux-plugins/tmux-continuum'
set -g @continuum-restore 'on'
```

**Features:**
- Auto-save every 15 minutes
- Auto-restore on tmux start
- Works with tmux-resurrect

## Shell Extensions

### Essential (HIGH Priority)

#### 1. fzf Shell Integration
**Purpose:** Fuzzy finding in shell
**Status:** NOT CONFIGURED (fzf installed)

**Installation:**
```bash
# Add to .bashrc
source /usr/share/doc/fzf/examples/key-bindings.bash
source /usr/share/doc/fzf/examples/completion.bash
```

**Provides:**
- Ctrl-R: Fuzzy history
- Ctrl-T: Fuzzy file find
- Alt-C: Fuzzy cd

---

#### 2. fzf-git.sh
**Purpose:** Git-aware fuzzy finding
**Repository:** junegunn/fzf-git.sh
**Status:** NOT INSTALLED

**Installation:**
```bash
# Download
curl -O https://raw.githubusercontent.com/junegunn/fzf-git.sh/main/fzf-git.sh
# Source in .bashrc
source ~/fzf-git.sh
```

**Provides (with Ctrl-G prefix):**
- Ctrl-G b: Branches
- Ctrl-G c: Commits
- Ctrl-G t: Tags
- Ctrl-G r: Remotes
- Ctrl-G s: Stashes
- Ctrl-G f: Changed files

---

#### 3. zoxide
**Purpose:** Smart directory navigation
**Package:** Available via apt/cargo
**Status:** NOT INSTALLED


**Shell Integration:**
```bash

```

**Usage:**
- `z dir`: Jump to matching directory
- `zi`: Interactive with fzf
